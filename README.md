# DSDV-Project---RETAIL-DASHBOARD

This project is a data visualization dashboard built on top of the **Customer Shopping Dataset** (from Kaggle).  
The goal is to explore, filter, and visualize retail / shopping behaviors, such as number of purchases, trends over time, customer segments, etc.

## Dataset

- Source: [Customer Shopping Dataset (Kaggle)](https://www.kaggle.com/datasets/mehmettahiraslan/customer-shopping-dataset)

## ⚙️ Requirements

Before running the project, make sure your system has:

### Node.js (v18 or higher)
Download: https://nodejs.org/

Check installation:

```bash
node -v
```

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Marceline4972/DSDV-Project---RETAIL-DASHBOARD.git
cd DSDV-Project---RETAIL-DASHBOARD
```
2️⃣ Install dependencies
```bash
npm install
```

3️⃣ Start the development server
```bash
npm run dev
```

The project will run at:
```bash
http://localhost:5173
```
